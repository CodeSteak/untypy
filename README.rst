untypy
======

A library for dynamic type checking in python. (soon)

Current Features:
-----------------
- Simple Types like int, str, bool ...
- Callable (e.g. lambda)
- None
- Any
- Literal
- Tuple

Partially Implemented
---------------------
- Union (Only Simple Cases)
- list (No Callables etc. inside)
